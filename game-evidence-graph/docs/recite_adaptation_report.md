# ReCITE Adaptation Report

Run:

```bash
game-evidence recite report --repo-path /path/to/ReCITE --out docs/recite_adaptation_report.md
```

This project reuses ReCITE-style ideas for strict JSON outputs, conservative formatting correction, graph precision/recall/F1, and LLM-as-judge evaluation. It does not reuse domain assumptions that papers contain complete ground-truth causal graphs.
